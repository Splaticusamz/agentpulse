# Revenue Strategy

## Market Gap

The AI agent ecosystem (MCP, ACP, LLM APIs) has **zero dedicated monitoring tools**. Traditional uptime monitors (Pingdom, UptimeRobot) don't understand agent protocols.

## Pricing Tiers

### Free — $0/mo
- 3 monitored endpoints
- 5-minute check intervals
- Public status pages
- Uptime badges (SVG)

### Pro — $9/mo
- 25 endpoints
- 1-minute checks
- Email + webhook alerts
- Private status pages
- API access
- 90-day history

### Team — $19/mo
- 100 endpoints
- Multi-region checks
- Custom domains for status pages
- Webhook integrations
- Priority support
- 1-year history

## Go-to-Market

1. **Week 1:** Launch free tier, post on AI/MCP communities
2. **Week 2:** SEO optimization, badge adoption drive
3. **Week 3:** Pro tier launch with Stripe integration
4. **Target:** 50 free users → 10 Pro conversions in first month

## Synergies

- **ACP Watchtower** → AgentPulse pipeline: validate your manifest, then monitor it
- **Badge virality:** Every README badge = free marketing
